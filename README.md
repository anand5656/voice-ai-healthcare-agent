# Voice AI Healthcare Assistant

Realtime multilingual AI healthcare assistant using:
- FastAPI
- Deepgram
- OpenAI
- Next.js
- WebSockets

## Features

- Realtime voice transcription
- AI appointment booking
- Multilingual support
- Memory system
- Live transcripts
- Professional animated dashboard
- Conversation history

## Tech Stack

### Frontend
- Next.js
- Tailwind CSS
- Framer Motion

### Backend
- FastAPI
- Deepgram SDK
- OpenAI SDK

## Setup

### Backend

```bash
cd backend
pip install -r requirements.txt
